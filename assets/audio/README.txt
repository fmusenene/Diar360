Background Music for Diar 360 Commercial
=========================================

Add your royalty-free background music file here:

  Filename: background.mp3
  Format:   MP3
  Style:    Corporate, inspirational, or ambient (subtle, non-distracting)

Recommended sources for royalty-free music:
- Pixabay Music: https://pixabay.com/music/
- Mixkit:       https://mixkit.co/free-stock-music/
- Bensound:     https://www.bensound.com/ (some free with attribution)

When online, the commercial uses a default track from Mixkit.
When offline (e.g. USB), add background.mp3 here and run COPY_TO_USB.
